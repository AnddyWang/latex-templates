LaTeX is a typesetting program used to produce high quality technical documents. Formatting LaTeX documents is difficult and pre-built templates are often difficult to modify. The following are LaTeX templates I'm curating and creating that strive to be minimalist, which are automatically generated from the bamos/latex-templates repository on GitHub. See here for a further introduction to LaTeX. The Makefile's use btford/write-good][write-good] to error and not build the document if the content contains the passive voice and throw a warning (but still build the document) for other errors.
